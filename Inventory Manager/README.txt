This project was an OOP project created in Python, although OOP is not best suited for Python the intention was to get us use to OOP as we were moving to Java which has
better adaptability to OOP. 
We were tasked to create an inventory manager that could perform the following functions;
- Read shoe data ( the country they have been delivered from, product code, name of product, cost and quantity)
- Capture shoe data 
- View all;
  -tabulate was used to display all the inventory information
- Restock Shoes ( add a new shoe )
- Value per item 
- Highest quantity
